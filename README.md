# House Finder (name TBD)

Easily find housing

## Requirements

- [Node.js](https://nodejs.org/en/) 6+

## Installation

```shell
npm install
```

OR (if you have yarn installed)

```shell
yarn
```

## Development

To start the server in Development mode:

```shell
npm run start:dev
```

OR

```shell
yarn start:dev
```

To start the server in Production mode (for deployment):

```shell
npm run start
```

OR

```shell
yarn start
```
